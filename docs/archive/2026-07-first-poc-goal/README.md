# First PoC Goal archive (July 2026)

This directory preserves the documentation snapshot used to implement and accept the first Project Tavern seven-day PoC engineering Goal. The final post-Goal documentation checkpoint was `c591a66443374f68a37f375ad219bcba43711280`.

## Status

Everything below this directory is **historical and non-authoritative**. It remains available for design archaeology, rationale, and evidence lookup, but it is not:

- a current execution plan or required reading set;
- a source of active API or gameplay contracts;
- a reason to preserve provisional PoC behavior or balance;
- an instruction to recreate phase ordering, task commits, materialization attestations, exact toolchain pins, offline/clean-tree gates, remote balance evidence, or release checkpoints.

Use the [active documentation map](../../README.md) for current development.

## Snapshot contents

- `root/` — the former root `AGENTS.md` (stored as `AGENTS.snapshot.md` so it cannot act as live repository instructions), `README.md`, and `CONTRIBUTING.md`;
- `docs-index.md` — the former `docs/README.md`;
- `engineering/` — Goal entrypoint, execution protocol, plans, checkpoints, and detailed specifications;
- `runbooks/` — runbooks written for the Goal-era delivery chain;
- `poc/` — the provisional seven-day rules, balance, content, and reference strategies;
- `art/` and `design/` — the former visual-pack and game-design baselines.

Files were moved as a snapshot rather than rewritten. Their old relative links may no longer resolve from the archive location, and archived files are intentionally excluded from active documentation link maintenance.
